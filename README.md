# Environment Metrics Link

This project is a proof of concept for environment metrics tracking using Zephyr RTOS to send data to the cloud and allow ML workflows.

Refer to the [Infrastructure & Operations Guide](./docs/03-infrastructure.md) for details on deployment and secure dashboard access.
